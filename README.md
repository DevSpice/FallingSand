# FallingSand
Falling sand project for C++ University of Chicago CS Master's program

To run:
(1) Download and install the latest version of Raylib. (Set-up video here: https://www.youtube.com/watch?v=m8sbsDM93tI)
(2) Clone our GitHub repository to your computer, from the last_working_main branch. (https://github.com/DevSpice/FallingSand.git)
(3) Run main.exe, OR to run via VSCode:
    (a) Open our project repo folder, and navigate to main.cpp
    (b) Perform the command "Ctrl+F5"
    (c) Allow the project to load, and proceed to playing with our Falling Sands on your machine!

Enjoy! :-)
